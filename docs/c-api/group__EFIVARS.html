Writes buffer contents to storage.

Note that as writing operations are asynchronous, to check if it completely succeded, one needs to register a callback that will inform writing result.

Parameters
namename of property. It will create a file on filesyste with this name.
blobblob that will be written
cbcallback to be called when writing finishes. It contains status of writing: if failed, is lesser than zero.
datauser data to be sent to callback cb

return 0 on success, a negative number on failure
