Detailed Description

Soletta vector is an array that grows dynamically. It's suited for storing a small set of contiguous data. Its dynamic resize might shuffle the data around, so pointers returned from sol_vector_get() and sol_vector_append() should be considered invalid after the vector size is modified.

For storing pointers, see sol_ptr_vector().
