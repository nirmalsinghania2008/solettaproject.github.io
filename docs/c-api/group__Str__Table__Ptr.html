Retrieves the value associated with a given key from the string/pointer table.

Searches the table table for key string and return its pointer, if key isn't found, the pointer fallback is returned.

Parameters
table_ptrString/pointer table
keyKey to search
fallbackFallback pointer
Returns
If key is found, return it's value, otherwise fallback is returned.
