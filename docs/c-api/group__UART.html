Perform a UART asynchronous transmission.

Parameters
uartThe UART bus handle
txThe output buffer to be sent
lengthnumber of bytes to be transfer
tx_cbcallback to be called when transmission finish, in case of success the status parameter on tx_cb should be equal to length otherwise an error happen during the transmission
datathe first parameter of tx_cb
Returns
true if transfer was started
Note
Caller should guarantee that tx buffer will not be freed until callback is called.
