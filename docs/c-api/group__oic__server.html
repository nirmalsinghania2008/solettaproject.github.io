const void *  data 
)

Send notification to all clients in observe list.

Send a notification packet with data filled by fill_repr_map callback to all clients that have registered in the resource as an observer.

Parameters
resourceThe target resource
fill_repr_mapCallback to fill notification data. Callback parameters are data, the user's data pointer, oic_map_writer, the write handler to be used with sol_oic_map_append to fill the notification data
dataPointer to user's data to be passed to fill_repr_map callback
Returns
True on success, false on failure.
