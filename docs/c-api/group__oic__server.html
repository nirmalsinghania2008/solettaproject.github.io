int sol_oic_server_init ( void  )

Initialize the oic server.

The first call of sol_oic_server_init will initialize the oic-server and any other following calls will increment the oic-server reference counter. After using the oic-server, sol_oic_server_release() must be called to decrement the reference counter and to release the resource when possible.

This function must be called before calling any other sol-oic-server function.

Returns
0 on success. A negative integer on errors.

void sol_oic_server_release ( void  )

Release the oic server.

Decrement the reference counter of the oic-server. When last reference is released all resources associated with the oic-server are freed.
