Detailed Description

A sol_buffer is a dynamic array, that can be resized if needed. It grows exponentially but also supports setting a specific size.

Useful to reduce the noise of handling realloc/size-variable manually.

See also sol-arena.h if you are allocating multiple pieces of data that will be de-allocated twice.
