Detailed Description

Structure to keep a single oic-map's field.

Use this structure to read fields using an sol_oic_map_reader and macro SOL_OIC_MAP_LOOP() or to write fields using an sol_oic_map_writer and function sol_oic_map_append.

See also
sol_oic_map_append()
SOL_OIC_MAP_LOOP()
